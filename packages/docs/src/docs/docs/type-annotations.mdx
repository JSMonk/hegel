# Type Annotations

---

Type Annotations it's the most valuable part of Hegel.

Type Annotation is an ability to declare values that are valid for the variable. Lets explore this ability.

First of all, lets write a simple function in pure JavaScript.

```javascript
function mul(a, b) {
  return a * b;
}
```

So, you can apply this function with different values:

```javascript
mul(42, 42); // 1764
mul(42, "42"); // 1764
mul(true, "42"); // 42
mul(class User {}, "42"); // NaN
```

As you can see, applying this function with different data types sometimes will return a valid result.
But, you always want to get a valid result. So, you can add types.

```typescript
function mul(a: number, b: number) {
  return a * b;
}
```
